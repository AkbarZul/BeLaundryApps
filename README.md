# BeLaundryApps 

## About The Project
 Dibuat dengan framework javascript yaitu React Native untuk frontend, ExpressJs untuk backend, dan MYSQL untuk database.

 #### Built With

- [ReactNative](https://reactnative.dev/)

- [ExpressJS](https://expressjs.com/)

- [MySQL](https://www.mysql.com/)
